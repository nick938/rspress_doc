草台班子 第一部分   
因为是有些同事先到的，然后我们三个武汉的是后到的。就直接住的是博尔塔拉大酒店。当时是一个人一间房。后来经理来了，说公司有规定。出差必须两个人住一间房。后来就和同事住到一间房里了。那个时候的报销也是非常慢。最开始的一笔报销。过了四个月才到我手里。当时还是用的信用卡。住酒店的钱都是我们自己现行垫付后再报销。导致当时资金流有点紧张。每个月多还200多的手续费。shit。后来两个人住一间房有很多不方便的事情。跟经理说了，然后也分开住了。真是傻逼公司。没把我们当人。正式员工都是一人一间房。而且正式员工出差补贴120一天。外包80一天。这些只明面上的。私底下外包还要受不少歧视。

因为去武汉是给新疆公路局做项目的。压力是要比在武汉大一些的。武汉人多。这边人少。当时给我累够呛。因为当时就我一个前端。说有功能需要定制。然后几天之后要。做前端。加上测试。加上发版本。全是我一个人。给我干眼症都干出来了。后来买眼药水，去武汉做眼部spa。才缓解了。
